# 📚 BCA Study Planner

A web-based study planner built for **BCA (Bachelor of Computer Applications)** students affiliated with **CCS University**. This project helps students organize their academic life — from subjects and notes to timetable, exams, and calendar.

---

## 🌐 Live Pages

| Page | Description |
|------|-------------|
| `BcaStudyPlanner.html` | Main Dashboard |
| `subject.html` | Subject Manager |
| `notes.html` | Notes Section |
| `timetable.html` | Weekly Timetable |
| `exam.html` | Exam Tracker |
| `calendar.html` | Academic Calendar |
| `BcaStudyPlanner.css` | Styling & responsive design |

---

## 🛠️ Tech Stack

- **HTML5** — Structure & layout
- **CSS3** — Styling & responsive design
- **JavaScript (Vanilla)** — Logic & DOM manipulation
